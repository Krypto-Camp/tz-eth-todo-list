### How To Start | 如何開始

```bash
npm install
# 使用 ganache 啟動本地鏈
launch your localRPC server by Ganache
npm run dev


# if u want testing ur contract by write testing
# u can
npm run test
```


### Hint: Check your ganache port is the same as truffle-config.js | 確保你的 ganache port 跟 truffle-config.js 的檔案裡設定的是一致的
